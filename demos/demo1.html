CustomStyle.css Overview

What are some functions?

There are many functions, these include: <hr>, <a>, <b>, <i>, <q>, and <pre>

<hr> defines a horizontal rule
<a> defines a hyperlink
<b> defines bold text
<i> defines italic text
<q> defines quotes
<pre> defines codeblocks
